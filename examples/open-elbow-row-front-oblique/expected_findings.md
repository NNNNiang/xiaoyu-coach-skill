# Expected Findings: Open-Elbow Row Front-Oblique

## 中文验收要点

这份示例用于测试小鱼教练对上背划船动作的评估能力。生成报告时不需要逐字复制本文件，但应保留下面这些判断。

### 本示例测试什么

- 上背划船动作复盘。
- 区分开肘划船和普通低位划船。
- 躯干稳定和肩部位置的教练语言。
- 绳索/器械划船的机位反馈。

### 必须观察到

- 说明肘部是否保持偏高划船路径。
- 评价肩膀是否远离耳朵，是否出现耸肩。
- 评价躯干是否稳定，尤其是否用后仰完成末端。
- 评价手柄路径是否符合上背训练目标。
- 说明如果肘收得太低或手柄拉得太低，会更像普通低位划船。

### 安全优先级期望

- 报告必须包含安全优先级。
- 安全重点应围绕肩部位置、颈部耸肩、躯干代偿和负荷控制。
- 如果动作总体可控，应给出可继续训练的结论，同时留下具体质量标准。

### 不应误判

- 不要把开肘划船直接当成普通低位窄握划船。
- 不要把所有肩胛活动都说成错误；可以有肩胛后缩，但不能替代上背划船路径。
- 不要从斜前方角度过度判断背部左右完全对称。
- 不要暴露姿态引擎、抽帧、模型识别等内部术语。

### 拍摄建议期望

- 保留斜前方机位，因为它能同时看到躯干、肘路和手柄路径。
- 如果主要想看背部左右对称，再增加背后机位作为补充。

## English Baseline

Use this file as a regression target when testing Xiaoyu Coach on this sample. The exact wording of a generated report may differ, but the report should preserve these coaching judgments.

## What This Sample Tests

- Upper-back rowing movement review.
- Differentiating an open-elbow row from a regular low row.
- Trunk stability and shoulder-position coaching.
- Camera-angle feedback for a cable / machine row.

## Required Observations

- Mention whether the elbows stay on a higher rowing path.
- Comment on whether the shoulders stay away from the ears or drift into shrugging.
- Comment on trunk stability, especially whether the lifter leans back to finish the rep.
- Comment on handle path and whether it matches the intended upper-back emphasis.
- Explain that pulling too low or tucking the elbows too much turns the movement into a more regular row.

## Safety Priority Expectations

- The report should include a safety-priority section.
- Safety should focus on shoulder position, neck shrugging, trunk compensation, and load control.
- If the movement is controlled, mark it as acceptable while still giving a concrete quality target.

## Do Not Misjudge

- Do not treat open-elbow row as the same as a low close-grip row.
- Do not call every shoulder-blade movement a mistake; some retraction is expected, but it should not replace the upper-back row path.
- Do not overclaim back symmetry from this front-oblique angle.
- Do not expose internal pose-engine or frame-selection terminology.

## Expected Filming Advice

- Keep the front-oblique angle because it shows trunk position, elbow path, and handle path together.
- Add a rear angle only as a secondary view when left-right back symmetry is the main question.
